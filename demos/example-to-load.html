Syntax Reference

Radial Gradient Syntax

Syntax Example Description
Basic radial-gradient(red, blue) Circle from center
Shape radial-gradient(circle, red, blue) Force circle shape
Size radial-gradient(circle closest-side, ...) Size to closest side
Position radial-gradient(circle at top right, ...) Position gradient center
Combined radial-gradient(ellipse farthest-corner at 70% 30%, ...) All parameters

Conic Gradient Syntax

Syntax Example Description
Basic conic-gradient(red, blue) Sweep from 0deg
Starting angle conic-gradient(from 90deg, red, blue) Start from 90deg
Position conic-gradient(from 0deg at 50% 50%, ...) Position center point
Hard stops conic-gradient(red 0deg 90deg, blue 90deg 180deg) Pie chart sections
Use Cases:
  • Radial: Spotlights, vignettes, buttons, loading indicators, focus effects
  • Conic: Pie charts, color pickers, loading spinners, progress indicators, patterns
